For the undecisive movie lover in all of us, Films Plus Friends is your one stop shop for planning a movie night. We will generate 5 movies for you based on rating and genre, you can either keep or swap out to create the ultimate movie playlist. After you pick your movies, invite your friends to vote on what you watch!

It's simple, it's fun, and you might even discover a new favorite movie!

Libraries & API's

Firebase

User data and Movie selections are sent to Firebase to be queried by later in the process. Each session gets a unique Access ID that is sent to the hosts and guests to allow them to vote on the Movies selected.

Read more about Firebase

API

TMDB - The Movie Database:

TMDB has a large and comprehensive library of Movies and includes severable queriable filters when connecting to the DB. Allowing us to generate random Movies based on user input and movies based on user search.

Read more about TMDB

EmailJS:

Email.js allows users to use their API to handle sending emails as opposed to installing the component on your site. The API allows you to create templates that can include dynamic variables and reference the template ID right in the API call so keeps formatting clean and easy.

Read more about EmailJS

JS Library

Animate.css:

Animate.css is an animations library that allows you to target elements with animations by adding a class similar to Bootstrap. This library handles all the animations on our site.

Read more about Animate.css
